# VR Hangman Game

This is a VR Hangman game built using Unreal Engine with SteamVR support.

## Project Structure

- **Source/**: Contains the game source code.
- **Content/**: Contains assets like models and textures.
- **Config/**: Contains configuration files for the game.

## Getting Started

1. Clone the repository.
2. Open the project in Unreal Engine.
3. Set up the SteamVR plugin.
4. Build and run the project.
